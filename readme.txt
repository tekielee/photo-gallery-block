Photo Gallery Block.

== Description ==

With admin page option to add Unsplash access key and search terms. API access key, terms, and images are
saved in the wp_option table. The options data will pulled out as default settings. The images will be display
in the frontend. It also is displayed in the page itself.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/photo-gallery-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
